# umi project

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

### 知识点

- Less
- umijs
- React


## React Hook

> Hook不能在class组件中使用。你可以创建你自己的hook来复用不同组件之间的状态逻辑。

> 通过使用 `Hook`,你可以把组件内相关的副作用组织在一起(例如创建订阅及取消)

> 自定义`Hook`解决了以前在React组件中无法灵活共享逻辑的问题。
